Issue #14 — The tools are splitting

Apr 7 → 14
The big story

Cursor 3 ships — and it's not an IDE anymore. It's an agent control room.

Cursor 3 launched April 2 with a completely new interface built from scratch around managing fleets of agents — not writing code. The Agents Window lets you run multiple agents in parallel across local, cloud, and remote environments. Design Mode lets you click directly on browser UI elements to give agents precise visual feedback, which is massive for designers prototyping in code. The top Hacker News comment: "I still want to code, not vibe my way through tickets." Cursor's response: the IDE still exists. Both are telling the truth.

Cursor Blog Changelog TechCrunch 💰 $2B ARR 🏢 100% adoption at Datadog 📅 Apr 2

Figma opens write access to AI agents

Agents can now create and modify real design assets via MCP — using your components, variables, and tokens. They shipped Skills: markdown files that teach agents how your team works. Free during beta, 10+ MCP clients supported. The question isn't if agents will design — it's who writes the rules they follow.

Figma Blog Bitovi test 📡 171 mentions/mo

Lovable hit $200M ARR in one year

The vibe-coding platform went from zero to $200M ARR faster than almost any software company in history. Their pitch: "minimal lovable product" replacing MVP. Designers and PMs are building production apps from prompts — no developer, no design queue. Influencer marketing is 10x bigger for them than paid social. The growth story was the biggest Lenny's Podcast episode this quarter.

Lenny's Podcast 💰 $200M ARR 📅 1 year

Our read: Two different bets on the same future. Cursor says developers manage agent fleets. Lovable says non-developers skip the fleet entirely. Both are right — for different people. The designer's job is increasingly to define what good looks like clearly enough that either path works.

Only 31% of designers use AI for core work

Humbl Design's 2026 report with real adoption data: only 31% of designers use AI for core design work vs 59% of developers. AI won't replace designers — but it will "wipe out pixel-pushers." Designers who only care about making things look clean are no different from AI.

Humbl Design 📊 data report 📅 2026

The pattern across all of these: the role isn't shrinking, it's splitting. One track leads to strategy, evaluation, and system-level thinking. The other track — execution without judgment — is what AI automates. Every source we read this week pointed to the same conclusion from a different angle.

Tools & workflows

Onlook: "Cursor for designers"

Drag-and-drop, rearrange, and scale elements directly in your actual React app — like Figma but editing real code. No terminal needed. One designer called it "Figma and v0 fused into a devilish combo." The boundary between design tool and code editor is melting.

The bottom line

The design tool landscape fragmented this week. Cursor built an agent control room with visual design feedback built in. Lovable proved non-developers can ship production apps. Onlook merged Figma's editing metaphor with real codebases. And Figma opened its canvas to agents while fighting a credit pricing revolt.

The thread connecting all of it: the designer's value is moving from production to judgment. Who decides what's worth building. Who writes the rules agents follow. Who evaluates whether the output is good enough. That's not a smaller job — it's a harder one.
